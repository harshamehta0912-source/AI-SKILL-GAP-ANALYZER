# 🤖 AI Skill Gap Analyzer for Manufacturing

An AI-powered web application that analyzes an employee's skills, identifies skill gaps, recommends learning resources, and generates a skill assessment report.

---

## 🚀 Features

- 👨‍💼 Employee Skill Assessment
- 📊 Skill Match Score Calculation
- 🔍 Missing Skill Detection
- 📚 Course Recommendations
- 🗄️ SQLite Database Integration
- 📄 PDF Report Generation
- 📈 Employee Dashboard
- 🌐 Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- ReportLab (PDF Generation)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/harshamehta0912-source/AI-SKILL-GAP-ANALYZER.git
```

2. Open the project folder

```bash
cd AI-SKILL-GAP-ANALYZER
```

3. Install dependencies

```bash
pip install flask reportlab
```

4. Create the database

```bash
python create_db.py
```

5. Run the application

```bash
python app.py
```

6. Open your browser

```
http://127.0.0.1:5000
```
